# StarAI Practice Client 1.2.3

봇-맵 호환성 필터를 추가로 보강한 핫픽스 릴리즈입니다.

## 수정 사항

- 최근 전적에서 1분 이내에 끝나고 정상 승패가 남지 않은 봇-맵 조합을 래더/랜덤/명시 선택 후보에서 제외했습니다.
- 투혼 계열에서 즉시 종료되거나 튕길 수 있는 `KillAlll`, `Iron bot`, `XIAOYICOG2019`, `Zia bot` 조합을 차단했습니다.
- `Crazyhammer` + `(4)Empire of the Sun`, `McRaveZ` + `(4)La Mancha1.1`, `PurpleWave` + `(4)Polypoid 1.65` 조합을 차단했습니다.
- 투혼에서 정상 동작하는 기존 봇 후보는 계속 선택할 수 있습니다.

## 설치 방법

1. `StarAI-PracticeClient-1.2.3-win-x64.zip`을 다운로드합니다.
2. ZIP 파일의 압축을 풉니다.
3. 압축을 푼 폴더에서 `install.cmd`를 실행합니다.
4. 설치 후 `C:\starai\Start-StarAI-PracticeClient.cmd`로 실행합니다.
